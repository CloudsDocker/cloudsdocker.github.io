# Shortcuts

## Open files by name

To navigate to a class, file or symbol with the specified name:

On the main menu, point to Navigate, and then choose Class, File, or Symbol respectively, or use the following shortcuts:
Class: ⌘O
File (directory): ⇧⌘O
Symbol: ⌥⌘O
